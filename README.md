## dbの作成方法
ターミナルでpython3を開く　　
<br>
- from app import app　
- from app import db　　
- with app.app_context():　　
<br>
    db.create_all()　　
<br>
instanceフォルダが作成され、その中にproject.dbがあることを確認する　　
<br>
- exit()　
<br>
python3から抜ける
