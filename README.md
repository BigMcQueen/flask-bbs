# flask-bbs
## 使用機器
Macbook Air(M1, 2020)
## 開発環境
venv(python:3.9.6)
## dbの作成方法
ターミナルでpython3を開く　　
- from app import app　
- from app import db　　
- with app.app_context():   db.create_all()　　
- instanceフォルダが作成され、その中にproject.dbがあることを確認する　　
- exit()　
- python3から抜ける
